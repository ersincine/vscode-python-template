# Visual Studio Code + Python

Do once:

- `python3 -m venv venv`

Then do:

- `source venv/bin/activate` (on Linux + macOS)
- `python -m pip install <package-name>`
- `deactivate`

We run only main.py:

- Run and Debug -> Start Debugging (Python: Module)
